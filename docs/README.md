## üé® Diagram Tools & Viewers

The architecture diagrams in this repository use [Mermaid](https://mermaid.js.org/) syntax, which is:
- Natively supported in GitHub Markdown
- Compatible with Azure DevOps Wiki
- Viewable in VS Code with Mermaid extensions
- Can be edited at [Mermaid Live Editor](https://mermaid.live)

### Viewing Diagrams
Simply navigate to any README with diagrams and GitHub will automatically render them.

### Editing Diagrams
1. Copy the Mermaid code block
2. Paste into [Mermaid Live Editor](https://mermaid.live)
3. Make your changes
4. Copy back the updated code
